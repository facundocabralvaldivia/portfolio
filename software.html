Software

Desarrollo y mantengo herramientas open source para investigación reproducible. Todos los proyectos están disponibles en mi GitHub.

  • Python v0.3.1 · pre-publicación

    MiPaquete

    Toolbox para análisis macroeconométrico

    Implementación modular de modelos VAR, BVAR, factor models y métodos de identificación estructural. Pensado para uso académico y aplicado, con énfasis en reproducibilidad. Incluye notebooks de ejemplo y datasets macroeconómicos pre-cargados.

    GitHub Documentación Tutorial .whl
  • Python v1.2.0 · estable

    nlp-econ

    Utilidades NLP para texto económico

    Pipelines pre-configurados para limpieza y vectorización de noticias económicas, reportes de bancos centrales y documentos de política. Soporte para múltiples idiomas y modelos transformer.

    GitHub PyPI
  • R v0.5.0

    fiscalR

    Análisis de escenarios fiscales

    Paquete R para construir y comparar escenarios fiscales de mediano plazo. Incluye módulos para sostenibilidad de deuda y ejercicios de stress.

    GitHub CRAN
  • Quarto Plantilla

    Plantilla de Investigación Reproducible

    Estructura de proyecto académico

    Plantilla Quarto con estructura clara para papers académicos: separación entre data, código, outputs; integración con Git; y pipeline de build automatizado.

    GitHub Demo
  • Dataset Actualización mensual

    Indicador Compuesto de Riesgo Institucional

    Datos abiertos · 2000–presente

    Indicador mensual construido a partir de noticias en múltiples idiomas, cubriendo más de 100 países. Disponible en CSV y vía API REST.
